💼 Meu Portfólio

Olá! Este repositório reúne o código do meu portfólio pessoal.

Desenvolvi este projeto com o objetivo de apresentar um pouco sobre mim, minhas habilidades, os projetos que venho desenvolvendo e facilitar o contato com recrutadores e empresas. Além de servir como uma vitrine do meu trabalho, este portfólio também representa minha evolução durante os estudos em Análise e Desenvolvimento de Sistemas.

🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

✨ O que você encontrará

- Uma breve apresentação sobre mim;
- Minhas principais tecnologias e habilidades;
- Projetos desenvolvidos durante meus estudos;
- Links para meu GitHub, LinkedIn e currículo;
- Layout responsivo para diferentes dispositivos.

🎯 Objetivo

Este portfólio foi criado para reunir, em um só lugar, os projetos e conhecimentos que venho adquirindo ao longo da minha formação. Meu objetivo é continuar evoluindo como desenvolvedor, aprender cada vez mais e contribuir com soluções que gerem valor.

📱 Responsividade

O site foi desenvolvido para funcionar de forma adequada tanto em computadores quanto em dispositivos móveis, buscando oferecer uma boa experiência de navegação em qualquer tela.

📬 Contato

Caso queira conversar sobre oportunidades, projetos ou trocar experiências, fique à vontade para entrar em contato pelos links disponíveis no próprio portfólio.
